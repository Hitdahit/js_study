/*
http:// -> protocol부분
      사용자가 서버에 접속할 때 어떤 방식으로 통신할 것인가
aaa.com  -> host(domain)
        인터넷에 접속된 컴퓨터
:3000  -> port
      한대의 컴퓨터 안에 여러개의 서버(개발자가 만든)가 있을 텐데
      이 때 해당 포트에 연결된 서버를 실행함.
main -> path
      컴퓨터안의 어떤디렉토리의 어떤 파일인지 지정
?id=HTML&page=12    ->query string
                  웹서버에게 전달하는 정보
                  항상 ?로 시작
                  &로 값들 사이를 나눔
                  =으로 값지정
즉
http://aaa.com:3000/main?id=HTML&page=12
*/
